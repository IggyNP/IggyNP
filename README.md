### What I do

I develop modern web applications with **JavaScript**, **React** and **Next.js**, turning designs into pixel-perfect, maintainable interfaces. I care about component architecture, semantic markup and writing CSS that scales.

- Building production-ready UIs with React, Next.js and Tailwind CSS
- Translating designs into responsive, accessible components
- Comfortable across the modern frontend toolchain and Git workflows

Beyond engineering, I bring a business perspective to product work — backed by an **MBA in International Management** and a **Master's in Digital Marketing & E-Commerce** — which helps me build features with the user and the business goal in mind.
